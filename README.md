# Library-Management-System

A Flask-based Library Management System designed to manage books and categories efficiently.

## Features

### Librarian Portal
- Add books
- Update books
- Delete books
- Manage categories
- Search books

### Reader Portal
- View books
- Search catalogue
- View categories

## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Python Flask

Database:
- MySQL

## Future Enhancements

- User authentication
- Borrow/Return system
- Fine management
- Dashboard analytics
