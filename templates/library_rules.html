Library Borrowing Guidelines

Please read the borrowing rules and library policies before borrowing any library material.

Eligibility

Borrow Limit

    {% if policy[2] == -1 %}
  • Members may borrow an unlimited number of books.
    • {% else %}
  • Maximum books allowed at one time: {{ policy[2] }}
    • {% endif %}
  • No additional books can be issued after reaching the borrowing limit.

Loan Period

    {% if policy[3] == -1 %}
  • Borrowed books do not have a fixed due date.
    • {% else %}
  • Every issued book is borrowed for {{ policy[3] }} day(s).
  • The due date is generated automatically during issue.
  • Books should be returned on or before the due date.
    • {% endif %}

Renewal Policy

    {% if policy[9] == 0 %}
  • Book renewal is not permitted.
    • {% elif policy[10] == -1 %}
  • Books may be renewed an unlimited number of times.
    • {% else %}
  • Books may be renewed up to {{ policy[10] }} time(s).
    • {% endif %}

Late Return

  • No fine is charged during the loan period.
  • Fine starts from the day after the due date.
  • The daily fine increases every {{ policy[6] }} overdue day(s).
  • Refer to the Fine Schedule below.

Damaged or Lost Books

  • Members are responsible for books issued to them.
  • Damaged or lost books may require replacement charges.
  • The librarian's decision regarding compensation is final.

General Rules

  • Handle library books carefully.
  • Do not write, underline or tear pages.
  • Return books through the library circulation desk.
  • Keep your membership information up to date.

Important Notice

• Members with unpaid fines may not be permitted to borrow additional books.
• Borrowing requests exceeding the permitted limit will be declined automatically.

• Library staff may refuse borrowing privileges if membership has expired.
